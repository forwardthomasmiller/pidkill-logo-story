Design Notes

  • Kill button uses circle + X icon (⊘ style), red tint, clearly distinct from delete (✕) and add (+)
  • Targets tab: Kill button between toggle and delete. Shows "N running" count. Disabled (dimmed) when no matching process is running
  • Browse tab: Kill button between PID and add (+). Tooltip shows exact PID being killed
  • Hover: brighter red background. Active/click: scale down feedback
  • After kill: row flashes red briefly, then auto-refreshes the process list
  • Kill result is logged to Kill Log, same as auto-kill
